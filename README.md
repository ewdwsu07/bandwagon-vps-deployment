## 搬瓦工部署全流程：买完到跑起来

### 第一步：注册账号、选套餐下单

进官网，选好套餐，点 Add to Cart 之后会让你选付款周期和机房。年付比月付能省两个月钱，建议直接年付。

机房的选择逻辑：CN2 GIA-E 套餐建议先选 DC6，后续不满意可以在 KiwiVM 后台一键迁机房，不用额外付费（部分机房有流量折算，迁之前看清楚规则）。

支付宝可以直接付款，填完账号信息、通过邮箱验证之后就进入支付流程。付款完成之后系统会发一封邮件，里面有订单确认信息。

### 第二步：进入 KiwiVM 后台，获取 SSH 信息

KiwiVM 是搬瓦工自研的控制面板，登进去之后 VPS 的一切操作都在这里做。

进入方式：登录搬瓦工账号后台 → Services → My Services → 点击对应 VPS 右侧的 KiwiVM Control Panel。

进去之后重点找这三个信息：

1. **IP Address**：你的服务器 IP
2. **SSH Port**：默认不是 22，是搬瓦工分配的随机端口，一般五位数
3. **Root 密码**：需要在 KiwiVM 里主动重置一次才会显示（Root password modification 菜单）

密码重置时 VPS 需要是开机状态。重置完之后密码只显示一次，复制好存起来，找不到就再重置一次。

### 第三步：SSH 连接进服务器

有了 IP、端口、密码这三个东西，就能连上服务器了。

**Windows**：推荐用 Termius（免费且跨平台），或者 Xshell。填 IP、端口、用户名 root，密码粘贴进去就连上了。

**Mac / Linux**：直接用终端，命令：


ssh root@你的IP -p 你的SSH端口


连上之后看到命令行提示符，说明已经在服务器里了。第一次进去建议先把系统更新一下：

bash
# Ubuntu/Debian
apt update && apt upgrade -y

# CentOS/Rocky Linux
yum update -y


### 第四步：根据用途部署具体服务

连上去了之后怎么用，取决于你想做什么。几个常见场景：

**建站（WordPress）**：推荐用宝塔面板，一条命令安装，图形界面管理 Nginx/Apache、数据库、SSL 证书，对不熟悉命令行的人友好很多。

**部署 AI 应用（n8n、Dify 等）**：基本都是 Docker 部署。先装 Docker，再用 docker-compose 拉起来。n8n 在 1GB 内存上勉强能跑，Dify 建议 2GB 以上。

**私有代理**：这个就不展开了，根据自己需求选协议，Xray/V2Ray 都是常见选择，装好之后正常使用就行。

## KiwiVM 的几个功能，新手容易忽略

KiwiVM 不只是看看 IP 用的，里面有几个功能挺有用：

**系统重装（Install new OS）**：支持 Ubuntu、Debian、CentOS、Rocky Linux 等主流发行版，现在还加了 Ubuntu 26.04。重装会清空所有数据，提前备份。

**机房迁移（Migration）**：CN2 GIA-E 套餐购买后可以在 12 个以上机房之间切换，发现某个机房延迟高直接迁过去，不用重新购买。

**在线 SSH（Root shell-interactive）**：本地 SSH 连不上的时候，KiwiVM 提供浏览器内的终端，作为救急用。

**快照（Snapshot）**：把当前系统状态保存下来，类似备份点。折腾系统之前可以先打一个快照，搞坏了还能回滚。

**流量统计**：可以看每月流量用了多少，避免用超。

## 几个容易踩的坑

连上 SSH 之后第一件事先改 root 密码。搬瓦工随机生成的密码挺长，容易出问题，换一个自己记得住的强密码。

机房选择上，不同运营商体验差别比较明显——电信用户 DC6/DC9 一般没问题，移动用户这两个机房也接入了 CMIN2，整体不差。联通用户通常对 CN2 线路感知最明显。购买前可以 ping 一下各机房测试 IP，看延迟再决定。

流量方面注意 KVM 套餐迁到 DC3 CN2 机房之后流量会变成三分之一，迁机房之前要看清楚规则。

## 常见问题

**Q：搬瓦工 VPS 买完多久能用？**

A：一般十几分钟内就开通了，系统默认安装好，进 KiwiVM 就能看到 IP 和端口。如果过了半小时还没开通，看一下邮件是否有额外验证要求。

**Q：搬瓦工支持支付宝付款吗？**

A：支持。结账时选择 Alipay 即可，会跳转到支付宝页面完成支付。银联也支持。

**Q：CN2 GIA-E 套餐和 KVM 套餐最大区别是什么？**

A：线路是最大区别。KVM 套餐走的是普通 CN2 GT 线路，CN2 GIA-E 走 CN2 GIA，后者在高峰期的稳定性和速度明显更好。如果主要用途是访问国内资源或者建站给国内用户访问，GIA-E 的体验会好一截。价格上，KVM 是 $49.99/年，GIA-E A 套餐是 $169.99/年，差不多三倍。

**Q：部署应用的时候内存不够怎么办？**

A：一是优化部署配置（比如 Dify 用存算分离架构把数据库外挂出去），二是直接升级套餐，搬瓦工支持在账户后台升级到更高配置。

**Q：搬瓦工有退款政策吗？**

A：有，购买后 30 天内可以申请全额退款。如果试用之后觉得不合适，在 30 天内联系官方客服提工单就可以。

**Q：限量版套餐和常规套餐的功能有区别吗？**

A：基本没有，功能一样，就是价格低很多。主要区别在于机房可选范围更少（通常固定一个或几个机房），而且缺货期长，有货就是赚到了。
